## 6. Revenue Stacking Economics

| Revenue Stack | Standalone IRR | Integrated Platform IRR | Delta |
|---|---|---|---|
| BESS only (merchant) | 8-12% | -- | -- |
| DC only (colocation) | 10-14% | -- | -- |
| Heat only | 4-8% | -- | -- |
| BESS + DC + Heat (integrated) | -- | 15-22%+ | +5-10% from synergies |

### Synergy Sources
- Shared grid connection (no incremental connection cost)
- Shared land/site (reduced per-MW cost)
- Heat revenue is incremental with zero fuel cost
- BESS provides grid services that benefit DC power quality
- Integrated offering differentiates from pure-play competitors

## 10. Key CH-NL Tax Flows

| Flow | Treatment |
|---|---|
| BV profit | NL CIT at 19%/25.8% |
| BV dividend -> AG | NL WHT 0% (CH-NL treaty, 25%+ holding) |
| AG receives BV dividend | CH Beteiligungsabzug: effective 0% |
| AG dividend -> investors | CH WHT 35%; reclaimable under treaty |
| AG -> CH individuals | Effective ~20-25% after partial reclaim |
| AG -> NL individuals | Treaty reduces to 15% |
| AG -> US entities | Treaty reduces to 5-15% |
