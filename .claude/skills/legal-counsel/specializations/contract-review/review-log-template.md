## File Naming Convention

`[YYYY-MM-DD]-[counterparty-name-lowercase-hyphenated].md`

Examples:
- `2026-02-17-stelia.md`
- `2026-03-01-coreweave.md`
- `2026-03-15-greenhouse-partner-a.md`

### DRAFT Format (In-Progress Reviews)

For reviews in progress (before Phase 7 finalisation), save as:
`DRAFT-[YYYY-MM-DD]-[counterparty-name-lowercase-hyphenated].md`

- Fields not yet completed should be marked `[PENDING]`.
- Decision field should read `PENDING — review in progress`.
- Update the DRAFT as each workflow phase completes.
- At Phase 7.6 (post-signature), rename the file by removing the `DRAFT-` prefix.
- DRAFT memos enable **session resumption**: if a new chat session detects a DRAFT memo for a counterparty, it can resume the workflow from where the previous session left off.
