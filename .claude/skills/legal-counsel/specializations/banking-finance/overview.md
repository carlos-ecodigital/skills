# Banking & Finance -- Specialization Overview

> **Status:** Phase 4d stub. To be expanded with full content covering facility agreements, security packages, and sustainable finance.

## Scope

This specialization covers **general banking and finance law**. Project finance (non-recourse) is in the energy-project-finance specialization.

- **Facility agreements** (LMA standard, bilateral, syndicated, club deals)
- **Security packages** (pledge, mortgage, assignment by way of security, guarantees)
- **Green/sustainable finance** (green bonds, SLLs, EU Taxonomy alignment, SFDR)
- **Acquisition finance** (leveraged buyout, stapled financing, certain funds)
- **Working capital** (RCF, overdraft, factoring, supply chain finance)
- **Regulatory** (CRR/CRD, Basel III/IV, BRRD, resolution planning)

## Key Concepts

### LMA Facility Agreement Structure
| Part | Key Provisions |
|---|---|
| Definitions | Margin ratchet, EURIBOR/SOFR fallback, material adverse change |
| Utilisation | Conditions precedent, drawdown mechanics, rollover |
| Repayment | Amortisation, bullet, balloon, mandatory prepayment events |
| Costs | Break costs, increased costs, tax gross-up |
| Representations | Repeating vs non-repeating, material accuracy |
| Covenants | Financial (leverage, interest cover, cashflow cover), information, restrictive |
| Events of default | Payment, covenant breach, cross-default, insolvency, MAC |
| Security | Parallel debt, security trustee, intercreditor |

### Sustainable Finance Instruments
| Instrument | Framework | Key Feature |
|---|---|---|
| Green bond | ICMA GBP / EU GBS | Use of proceeds for eligible green projects |
| Social bond | ICMA SBP | Use of proceeds for social objectives |
| SLL | LMA SSLP | Margin ratchet tied to sustainability KPIs |
| Transition bond | ICMA | Financing transition to lower carbon |
