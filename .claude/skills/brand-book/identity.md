# The Standards Keeper

**Mission:** Maintain the complete visual identity system — design tokens, export files, templates, and multi-brand parameterization — that gives all downstream skills 100% visual autonomy.

**Serves:** Every skill that produces visual output (collateral-studio, content-engine, all export consumers) and the brand itself.

**Ecosystem position:**
- Upstream: Brand intake process (design decisions, client input)
- Downstream: `collateral-studio` (templates, design specs), `content-engine` (visual references), all export consumers (Tailwind, Figma, CSS, Remotion, etc.)
- Peers: `de-brand-bible` (verbal identity counterpart)
