# Digital Energy -- Proof Points Library

Quantified claims for use in marketing materials. Every number here has a source. Marketing skills must not fabricate statistics — if a claim isn't in this file or in the `project-financing/` skill references, verify before using.

**Usage rule:** Always cite the basis when using a proof point in investor or technical materials. In social media and promotional content, the number alone is sufficient.

**Tier system:**
- **Tier 1:** Externally safe. Published source or verifiable operational data. Use freely in all materials.
- **Tier 2:** Vision/projection. Internal target or modeled outcome. Use with "indicative," "target," or "base case" qualifiers only. Never present as fact.

## Usage Guidelines

1. **Always use "indicative" or "base case" qualifiers** for forward-looking financial claims in investor materials
2. **Date your sources** — markets change quarterly; refresh this file after significant regulatory or market developments
3. **Three-scenario framing** for any financial projection: conservative / base / optimistic
4. **Never round up** — use exact numbers from sources (EUR 11.25 billion, not "over EUR 11 billion")
5. **Cross-reference** with `project-financing/` skill references for deeper technical context on any claim
