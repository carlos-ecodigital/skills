# Digital Energy -- Banned Phrases

These phrases are banned from all DE external communications. Use the replacement instead.

| Banned | Use Instead |
|---|---|
| Game-changing / revolutionary | Structurally differentiated / novel approach |
| Cutting-edge / state-of-the-art | Current-generation / production-ready |
| Synergy / synergistic | Integrated / co-located / complementary |
| Leverage (as verb) | Use / apply / deploy |
| Unlock potential | Monetize / develop / convert |
| Disruptive | Differentiated / non-traditional |
| Innovative solution | (describe what it actually does) |
| World-class | (prove it with data instead) |
| Stakeholders | Partners / investors / counterparties / (name the specific group) |
| Going forward / moving forward | (delete — just state the action) |
| At the end of the day | (delete) |
| It goes without saying | (if it goes without saying, don't say it) |
