# Proof Points Library — Ready-to-Use Claims for Collateral

Pre-formatted proof points organized by theme for direct insertion into presentations, one-pagers, proposals, and other deliverables. Each entry includes the claim, the number, the source, and suggested slide/section placement.

**Canonical source:** `de-brand-bible/references/proof-points.md` — this library reformats those proof points for collateral use. If a number here conflicts with the brand bible, the brand bible is authoritative.

**Rule:** Never fabricate a proof point. If you need a number that isn't here, flag it as "[TO BE CONFIRMED]" and note what data is needed.

## Usage Notes

1. **Slide titles:** Use the "Claim" column directly as slide titles for maximum impact
2. **Footnotes:** Use the "Source" column for slide footnotes in investor/technical materials
3. **Qualifier language:** For indicative figures, always add "indicative" or "subject to project-specific analysis"
4. **Three-scenario rule:** Any financial projection in a formal deliverable must show all three scenarios
5. **Update cycle:** Cross-check against `de-brand-bible/references/proof-points.md` quarterly — market data evolves
